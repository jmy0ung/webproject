마 실험과정

1. 에탄올 농도별 시료 제작

-0%: 마 5g + 증류수(D.W)20mL-> 코니칼 튜브에 넣음

-20%: 마 5g + 증류수(D.W)16mL + Et4mL-> 코니칼 튜브에 넣음

-40%: 마 5g + 증류수(D.W)12mL + Et8mL-> 코니칼 튜브에 넣음

-60%: 마 5g + 증류수(D.W)8mL + Et12mL-> 코니칼 튜브에 넣음

-80%: 마 5g + 증류수(D.W)4mL + Et16mL-> 코니칼 튜브에 넣음

-100%: 마 5g + Et4mL-> 코니칼 튜브에 넣음

45도 120rpm 2시간 항온수조에서 마 추출

-membrane 필터(0.45 마이크로 필터)로 15mL 코니칼 튜브에 냉동보관

2.폴리페놀 정량

-시료 0.2mL + 증류수 0.8mL + Folin 수용액(Folin 시약 5mL + 증류수 45mL) 1mL

-vortex 사용후 실욘 5분 반응

6% Sodium carbonate 1mL 첨가 후 vortex 사용

1시간 실온 반응

-725nm 흡광도 측정

3. 마 분말별로 시료 제작

-마 분말(1증 1포, 5증 5포, 9증 9포) 각 0.5g + Et:Dw=1:1로 총량 20mL

-50도 120rpm 2시간 항온수조에서 추출

-membrane 필터(0.45 마이크로 필터)로 15mL 코니칼 튜브에 냉동보관

4. 환원력

-시료 0.2mL + (시중판매) buffer solution(0.2M Sdium phosphate buffer) pH 6.86 0.5mL + 1% Potassium fefficyanid o.5mL(0.5g+ 49.5mL)

-50도에서 20분 반응

0.45 마이크로 필터로 필터링함

0.1% ferric chlorid 0.2mL(ferric chlorid0.05g + DW49.95mL) + ethanol 0.1mL 넣음

-700nm 흡광도 측정

5.ABTS 양이온 라디칼 소거능

-7.4mM ABTS 수용액 200mL(ABTS 0.7616g + 증류수 200mL) + 2.6mM Potassium persulfate 수용액 200mL(Potassium persulfate 0.1406g + 증류수 200mL)

-15시간 암실방치

-414nm에서 흡광도 측정 (O.D 1.5정도로 맞추기)-> 증류수로 희석

시료 0.15mL + ABTS 용액 3mL

실온 90분 반응

414nm에서 흡광도 측정

6. 아질산염 소거능

-희석시료 0.5mL + 1mM 아질산 나트륨 0.5mL + 0.1N HCl 4mL(pH1.2) 37도에서 1시간 항온수조에서 반응

-희석시료 0.2mL + 2% Acetic acid 1.25mL + Griess 시약 0.1mL(1% sulfanilic acid(0.1g+ 9.9mL 30%Acetic acid) + 1% naphthylamine(0.1g+ 9.9mL 30% Acetic acid)) vortex 사용

-520nm 흡광도 측정

7. DNS 법에 의한 환원당 정량

-시료 1mL + 증류수 2mL + DNS 시약 2mL (증류수-> 시료-> DNS 순)

-vortex 사용후 90도 중탕, 10분 반응

-575nm 흡광도 측정

8. 단백질 정량(lowry assay)

lowry C 제조-> lowry A 5mL[Sodium carbonate 1g + Sodium hydroxide 0.25g in 증류수 (총 50mL)] + lowry B CuSO4 0.05g in 증류수 (총 50mL)

-시료 0.5mL+lowry C 시약 3mL

-실온 15분 반응

-Folin 시약(증류수로 2배 희석하여 제조) 0.5mL 첨가

-실온 30분 반응

-750nm 흡광도 측정
